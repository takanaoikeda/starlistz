テキストのみでプレイリストを作成し、作成したプレイリストの売買を想定したWEBサービスです。

以前はプレイリストの売買サービスとして公開していましたが、今ではサービスを終了しポートフォリオとして公開しています。

※ポートフォリオ用に加工しているためサービス内で実際に売買をすることはできません。
